# Paradise Nursery by Rayen Haggui

**Paradise Nursery** is a React-based houseplant shopping application built as the final project for the IBM Full-Stack JavaScript Developer Professional Certificate (React Framework course).

## Project Name
Paradise Nursery - A houseplant e-commerce front-end application

## Features

- **Landing Page** - Background image, company description, and a "Get Started" button
- **Product Listing Page** - 18 unique houseplants across 3 categories (Tropical, Succulents & Cacti, Air-Purifying) with thumbnail, name, price, and Add to Cart
- **Shopping Cart Page** - View cart totals, adjust quantities, delete items, continue shopping, and checkout
- **Redux State Management** - Global cart state via Redux Toolkit
- **React Router** - Client-side navigation between pages
- **Responsive Design** - Works across desktop and mobile

## Tech Stack

- React
- Redux Toolkit + React Redux
- React Router DOM v7
- Create React App
- GitHub Pages (deployment)

## Getting Started

```bash
npm install

```
